# Meu Remédio Mapeado – Santo André

## Descrição

Projeto de extensão desenvolvido no município de Santo André (SP) com o objetivo de apresentar, em um mapa interativo, as unidades públicas de saúde e farmácias conveniadas que realizam a distribuição de medicamentos específicos, incluindo medicamentos para tratamento do tabagismo e medicamentos controlados.

---

## Funcionalidades

* Mapa interativo com unidades de saúde e farmácias conveniadas
* Marcadores diferenciados por tipo de atendimento
* Exibição de nome, endereço e horário de funcionamento das unidades
* Legenda explicativa para facilitar a compreensão do mapa

---

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* Leaflet.js
